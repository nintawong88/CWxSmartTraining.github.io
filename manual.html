การกระทบยอด (Reconciliation)

กระทบยอดวันที่ 1 (3 ก.ค.) และวันที่ 2 (6 ก.ค.) แยกกัน แล้วรวม Company Total

🪣

3-Bucket Reconciliation Model

🅰️
Bucket A — Match
นับแล้ว + มี SAP baseline
คำนวณ Variance = นับ − SAP
GR ถ้า Qty+ / GI ถ้า Qty−
🅱️
Bucket B — Found
นับแล้ว แต่ไม่มีใน SAP baseline
สินค้าที่ไม่เคยถูกบันทึก
GR ทั้งหมด (เพิ่มเข้า SAP)
🅲
Bucket C — Missing
มีใน SAP แต่ไม่ถูกนับ
Location นอกขอบเขต / Virtual
GI ทั้งหมด (ตัดออก SAP)
